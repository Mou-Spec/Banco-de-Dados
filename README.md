## Sobre o Projeto
Sistema de cadastro e consulta de pessoas desenvolvido em Java que permite gerenciar uma lista de pessoas em banco de dados local, com funcionalidades de listagem, busca por ID e busca por nome.

## Estrutura do Projeto

### Classes Principais

Pessoa - Representa a entidade pessoa com ID e nome.

BancoDeDados - Implementação principal da lógica de armazenamento e consultas.

ListaMain - Classe de testes com exemplos de uso das funcionalidades.

Main - Classe principal com menu interativo para operações do sistema.
